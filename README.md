# c_integrated_programs
# C Integrated Programs 🚀

This repository contains a collection of **important C programs for placement preparation**, combined into a **menu-driven system**. It is designed to help students practice core programming concepts and prepare for technical interviews.

---

## 📌 Features

* 🔢 Reverse a Number
* 🔍 Prime Number Check
* 🔁 Fibonacci Series
* ➗ Factorial (Recursion)
* 🔄 Palindrome Check
* 💡 Armstrong Number
* 🔀 Swap Two Numbers
* 🔎 Linear Search
* 📊 Bubble Sort
* 🧮 Matrix Addition
* 🔤 String Length Calculation
* 🔍 Binary Search
* ⚖️ Even/Odd Check

---

## 🧠 Concepts Covered

* Functions in C
* Loops (for, while)
* Conditional Statements
* Arrays & Strings
* Searching & Sorting Algorithms
* Recursion
* Matrix Operations

---

## 🛠️ How to Run

### 🔹 Compile the Program

```bash
gcc implementation.c -o program
```

### 🔹 Run the Program

```bash
./program
```
